Tritrig pipeline -- generation to analysis

The tritrig_slic_full_chain job (tritrig_dlic_full_chain_job.py) generates tritrig events using MadGraph5 and transforms the output from lhe to stdhep to then tag the mother particle and rotate the events to beam coordinates. Then, slic is use dto simulate the detector response, and empty bunches are added to simulate the signal pile-up. Lastly, the readout is simulated, hps-java is used to run the reconstruction, and hpstr is run to convert the reconstruction output files to root files and to run the analysis on those files.

Job parameters

This table contains parameters special to the tritrig_beam job. The general parameters are discussed on the examples main page.

param
steering_files readout and reconstruction steering files
config_files configuration files for recon and ana
output_files list of readout and reco slcio, as well as reco and analysis root output files
